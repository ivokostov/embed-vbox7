== Description ==

Директно вграждане на клипове от vbox7.com

Как работи можете да видите [тук](https://imgur.com/a/BElKDSv)

== Installation ==

1. Visit `Plugins > Add New`
2. Search for `Vbox7`
3. Activate Vbox7 embed support from your Plugins page.
